Method to return the Point that is nearest in angle to the given point p in the point set ps, in given direction (-1 for CW, 1 for ACW) Finds a point that has distinct coordinates wrt the given point, assumes it to be the nearest, then iterates over the points and updates the nearest point if orientation of the new point is in the given direction wrt to current nearest.

Parameters
point- Instance of the point to be taken as reference.
direction- Integer denoting the direction in which to find the nearest point (1 for ACW, -1 for CW)
Returns
- Point instance of the point nearest to the given point
